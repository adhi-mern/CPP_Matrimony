## Run Locally

Clone the project

```bash
  git clone https://github.com/AdithyakrishnaV/cppMatrimony.git
```

Go to the project directory

```bash
  cd cppMatrimony
```
Go to the project directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
